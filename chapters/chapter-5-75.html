The recover operation is called when the promise is failed, and it is used to replace the error by another value. You can see recover as the equivalent of a catch block in Java where you can handle an error. Here we simply provide a recovery value using a succeeded future, but you can also report a failed future in more advanced cases when there is nothing you can do to recover. The map operation transforms a value using a function, while flatMap composes with another asynchronous operation.
