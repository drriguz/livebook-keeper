Reactive extensions are an elaborated form of the observable design pattern [DesignPatterns]. They were first popularized by Erik Meijer in the Microsoft .Net ecosystem. Modern applications are increasingly composed of asynchronous event streams [Meijer12]. This is true not just on the server, but also in web, desktop and mobile clients. Indeed, we can think of graphical user interface events as a stream of events that an application has to respond to.
